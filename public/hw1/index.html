What is Machine Learning?
Machine learning is a subset of computer science which allows a program to utilize data to infer decisions, rather than explicit programming. Algorithms take in data as training, and use it to create a model which can be used to predict information about data similar to what the model was trained on. After training, you can provide a new input, and the output is generated by the model. For example, a predictive training algorithm will create a model capable of prediction of some feature of the training data.
